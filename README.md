# GoIT Algorithm Homework 10


# Алгоритми

**Жадібний** - вибирає найбільшу монету, O(n)  
**Динамічний** - гарантує оптимум, O(amount × n)

## Висновки

Для монет [50, 25, 10, 5, 2, 1]:

- Малі суми: обидва однаково швидкі
- Великі суми: жадібний в ~50000 разів швидший (100000: 0.000003с vs 0.150000с)

**Рекомендація:** використовуйте жадібний алгоритм

----------

# Монте-Карло/quad

## Результати

- Аналітичний (quad): 2.666667
- Метод Монте-Карло (N=100000): ~2.666 (результат варіюється)

## Висновки

- Метод quad дає точний результат з малою похибкою.
- Метод Монте-Карло дає наближення, яке покращується зі збільшенням N.
- Обидва методи підтверджують правильність обчислень.
